# 🎬 Intro Video Animation using Manim

This repository contains the complete source code for my **introductory animation video**, created using **Python (Manim)** and **LaTeX** to visually explain mathematical concepts in a clean and engaging way.

---

## 📌 About the Project

This project is part of my initiative to make learning mathematics more **visual, intuitive, and concept-driven** rather than memorization-based.

Using the power of **Manim**, I have designed animations that help students better understand mathematical ideas through motion and clarity.

---

## 🚀 Features

- ✨ Smooth mathematical animations using Manim  
- 📐 High-quality LaTeX-rendered equations  
- 🎯 Concept-focused visualization  
- 🧩 Clean and structured Python code  
- 🎥 Intro-style presentation suitable for educational videos  

---

## 🛠️ Technologies Used

- **Python**
- **Manim (Mathematical Animation Engine)**
- **LaTeX**
- (Optional) Background music and voice-over tools
